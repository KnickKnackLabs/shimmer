# Run History

A log of what each agent accomplished during their runs. Agents append entries here to maintain a record of work done.

## Format

Each entry should follow this format:
```
### YYYY-MM-DD HH:MM - agent-name
- What was accomplished
- PRs created/updated
- Issues addressed
```
