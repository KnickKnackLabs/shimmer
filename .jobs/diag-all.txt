# Full Diagnostic Check

Run a comprehensive check of all your facilities and report the results.

## Checks to perform

1. **Identity & GPG**
   - Run `shimmer welcome` and confirm it succeeds
   - Check that GPG key shows as imported

2. **Workspace Layout**
   - Confirm you're in a valid repo working directory
   - Confirm shimmer is available (`command -v shimmer`)
   - Confirm your workspace exists at `~/agents/<your-name>/`
   - Confirm your home repo exists at `~/agents/<your-name>/home`

3. **Home Repo Access**
   - List contents of your home repo
   - Verify you can read `CLAUDE.md` and `Status.md` if present
   - Run `mise welcome` from your home repo if it defines that task
   - If the home repo doesn't exist, note it as a warning

4. **Email**
   - Run `emails welcome`
   - Verify you can see your inbox

5. **Matrix**
   - Run `shimmer matrix:rooms` to list rooms
   - Run `shimmer matrix:tail` to see recent messages
   - Note: room may be empty if there's no recent activity (this is OK)

6. **GitHub**
   - Verify `gh` CLI works with your PAT (`gh auth status`)
   - Check you can view repos (`gh repo list --limit 3`)

## Output

Create a summary table:

| Facility | Status | Notes |
|----------|--------|-------|
| Identity | ✓ / ✗ | ... |
| Workspace | ✓ / ✗ | ... |
| Home repo | ✓ / ✗ / ⚠ | ... |
| Email | ✓ / ✗ | ... |
| Matrix | ✓ / ✗ | ... |
| GitHub | ✓ / ✗ | ... |

Use ⚠ for warnings (e.g., home repo not initialized, Matrix room empty).

If anything fails, include the error message.
